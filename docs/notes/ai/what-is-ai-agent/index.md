## 什麼是 AI Agent？

AI Agent 是一個能夠自主學習、自主思考、自主行動的智能體。它能夠感知環境、理解環境、並且根據環境的變化做出反應。AI Agent 通常是一個軟體程式，但也可以是一個機器人、一個無人機、一個自動駕駛汽車等等。
